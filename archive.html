02 · Concept
Identity asCataloging
System
  • Every object is a storage unit — assigned a position, identifier, and category
  • Everything has an ID — numbering is the primary communication tool
  • Visual = not decoration, but structure — typography navigates, grid orders
The identity is infrastructure, not expression.
Numbering becomes a central tool of communication, allowing the identity to operate as an indexing system across posters, digital interfaces, and physical storage.

06 · The Space — Playground & Classroom
Three
modes
of space
The three modes work on an escalating logic: from physical database to workshop environment to projected architecture that exists only in light.
01
Archive Mode
Shelving units with color-coded classification as a physical database. The body moves through the system the same way a cursor moves through a website.
02
Workshop Mode
Numbered stations 001 / 002 / 003 with tablets and materials. Each workstation is an indexed unit. A learning environment that is itself a catalog.
03
Stage Mode
A projector above the floor projects building layouts onto the surface. Participants sit inside buildings that do not yet exist. The classification transforms into space itself.

07 · Architect → Art Director
Architect → Art Director

This project didn't ask me to stop thinking like an architect — it asked me to apply that thinking to a different material. Classification systems, spatial grids, inventory logic, wayfinding: these are architectural problems.

Translating them into a visual identity system felt like a natural extension rather than a departure. The categories, color logic, and numbering system were designed in Figma. Exhibition and web visualizations were generated with AI to communicate spatial application — the system itself came first.

"The architect's instinct is to classify, order, and navigate. The identity system became the building."
